#Testing the Kanban Board

[![Actions Status](https://github.com/ElenaKazakova124/qa-auto-engineer-javascript-project-90/actions/workflows/hexlet-check.yml/badge.svg)](https://github.com/ElenaKazakova124/qa-auto-engineer-javascript-project-90/actions)

Цель проекта — тестирование приложения управления задачами на основе React Admin с помощью инструмента автоматизации тестирования Playwright. Тестирование охватывает проверку корректной работы основных функций приложения, таких как создание, редактирование, удаление и перемещение задач, а также авторизация пользователей.

##Описание проекта

В приложении реализованы следующие основные функции:

1. Управление задачами:
· Создание новых задач
· Назначение исполнителей
· Изменение статусов задач (например, "in progress", "draft", "done")
· Редактирование и удаление задач
· Перемещение задач между статусами на канбан-доске

2. Управление пользователями:
· Создание новых пользователей (для администраторов)
· Просмотр списка существующих пользователей

3. Управление метками:
· Создание и редактирование меток для задач (например, "bug", "feature")

4. Управление статусами задач:
· Создание и редактирование статусов задач, определяющих колонки на канбан-доске

###Технологии тестирования

Для тестирования приложения используется Playwright — современная библиотека для автоматизации тестирования веб-приложений. Playwright позволяет:

· Автоматизировать взаимодействие с интерфейсом приложения
· Эмулировать действия пользователя (клики, заполнение форм, навигацию)
· Проверять корректность работы различных функций
· Тестировать в нескольких браузерах (Chromium, Firefox, WebKit)
· Создавать скриншоты и видео для отладки

###Установка и запуск

Установка зависимостей

```bash
git clone https://github.com/ElenaKazakova124/qa-auto-engineer-javascript-project-90.git
cd qa-auto-engineer-javascript-project-90
npm install
```

Установка браузеров Playwright

```bash
npx playwright install
```

Запуск приложения в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

Запуск тестов

```bash
# Запуск всех тестов
npx playwright test

# Запуск конкретного тестового файла
npx playwright test tests/tasks.spec.js

# Запуск тестов в headful режиме (с отображением браузера)
npx playwright test --headed

# Генерация HTML отчета
npx playwright test --reporter=html
```
